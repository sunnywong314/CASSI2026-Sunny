****************
Useful Commands
****************

Here is a list of useful commands that you may run into over and over. 

.. list-table:: 
   :widths: 20 50
   :header-rows: 1

   * - Commands
     - Explanation
   * - rm
     - Remove item (add -r for directories)
   * - cp
     - Copy item (add -r for directories)
   * - mv
     - Move item
   * - ls
     - List items in directory (add -ltr for list all files in reverse time order)
   * - cd
     - Change directory (cd .. to go up one level)
